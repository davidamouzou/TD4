Qu'est-ce que Git ?

Git est un système de contrôle de version distribué, créé par Linus Torvalds en 2005. Il permet aux développeurs de suivre les modifications dans le code source durant le développement logiciel.

Fonctionnalités principales de Git

  • Suivi de versions : permet de suivre l'historique des modifications apportées aux fichiers.
  • Branches : permet de créer des branches pour développer des fonctionnalités en parallèle.
  • Fusion : permet de combiner différentes branches en une seule.
  • Distribution : chaque développeur a une copie complète du référentiel, y compris l'historique complet des modifications.
