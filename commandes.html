Liste des commandes Git

git init
Initialise un nouveau dépôt Git local.
git clone [URL]
Clone un dépôt distant à partir de l'URL spécifiée.
git status
Affiche l'état des fichiers dans l'arborescence de travail.
git add [fichier]
Ajoute le fichier spécifié à l'index (staging area).
git commit -m "[message]"
Enregistre les modifications dans l'historique du dépôt avec un message de commit.
git push
Envoie les commits locaux vers le dépôt distant.
git pull
Récupère et fusionne les modifications depuis le dépôt distant vers le dépôt local.
git branch
Liste les branches locales ou crée une nouvelle branche.
git checkout [branche]
Bascule sur la branche spécifiée.
git merge [branche]
Fusionne la branche spécifiée avec la branche courante.
git log
Affiche l'historique des commits.
git remote
Gère les connexions aux dépôts distants.
git fetch
Télécharge les objets et les références à partir d'un autre dépôt.
git diff
Montre les différences entre les fichiers modifiés et l'index.
